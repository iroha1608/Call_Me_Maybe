## 1.概要
---
このプロジェクトでは、自然言語のプロンプトを型付き引数を持つ構造化された関数呼び出しに変換するシステムを構築することで、大規模言語モデルにおける関数呼び出しの仕組みを紹介します。  
有効なJSON出力を保証するための制約付きデコードを実装し、わずか0.5Bパラメータのモデルでほぼ完璧な信頼性を実現することで、人間の言語とコンピュータが実行可能な操作との間のギャップを埋めます。
